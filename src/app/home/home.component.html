El trabajo se diseñó buscando cubrir los siguientes requerimientos

  • LISTADO DE NOMBRES

    Crear un arreglo con 3 nombres de personas, lo debes mostrar en pantalla en forma de lista con dos iconos/botones:

    1. Que nos permita eliminar el nombre seleccionado del arreglo.

    2. Que nos permita ver el nombre seleccionado una alerta de boostrap.

    Además, debemos de tener un pequeño formulario que nos permita agregar nombres a este mismo arreglo y que el input se limpie una vez se haya agregado el nuevo nombre.

  • LISTADO DE USUARIOS

    Crear un pequeño menú de navegación o pestañas que nos permita acceder a una nueva ruta llamada usuarios, en esta pantalla solo nos debe de mostrar una tabla que incluya las columnas: id, nombre, email, teléfono y ciudad.

    Para poblar la tabla de datos tenemos que consumir https://jsonplaceholder.typicode.com/users/ a través de angular HTTP, y mostrar solo los datos: id, name, email, phone y city respectivamente (ocultar de la tabla el usuario que tiene el id 3).

NOTA: Toda la maquetación de preferencia la debes hacer utilizando bootstrap. Puedes agregar características que explican los videos pero debes mantener las indicaciones anteriores.

PLUS: Que la tabla de usuarios tenga la opción para editar y eliminar el usuario seleccionado.
